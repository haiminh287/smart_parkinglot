# Vincom Center Parking

## Thông tin chung

- **Địa chỉ:** 72 Lê Thánh Tôn, Bến Nghé, Quận 1, TP. Hồ Chí Minh
- **Tọa độ GPS:** 10.7769°N, 106.7036°E
- **Số điện thoại:** (028) 3822-3333
- **Số tầng:** 1 tầng hầm (B1 / level=-1)
- **Tổng số slot:** 97 (72 ô tô V1 + 20 xe máy + 5 garage)

## Cấu trúc bãi

### Tầng B1 — Zone V1 (ô tô)
- 72 slot painted, code V1-01 → V1-72
- 4 hàng × 18 slot
- Slot width 2.5m, depth 5m
- Có divider cam rõ ràng, dễ nhận biết

### Zone Garage
- 5 slot garage khép kín (cho xe VIP, thuê tháng)
- Code G-01 → G-05
- Slot rộng 3m × 6m

### Zone Motorbike
- 20 slot xe máy
- Code M-01 → M-20

## Giờ hoạt động

- **Mở cửa:** 6:00 sáng hàng ngày
- **Đóng cửa:** 23:00 tối
- **Xe gửi qua đêm:** Được (nếu đặt package daily/weekly/monthly)
- **Lễ Tết:** Mở bình thường

## Phương tiện được hỗ trợ

✅ Ô tô ≤ 5 chỗ (sedan, SUV, hatchback)
✅ Xe máy
✅ Xe điện (V1-15, V1-30 có ổ sạc Type 2)
⚠️ SUV 7 chỗ lớn (slot garage only)
❌ Xe tải, xe khách
❌ Xe container
